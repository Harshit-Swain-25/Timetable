# Weekly Timetable

A simple Weekly Timetable project built using HTML. This project demonstrates how to organize and display a weekly academic schedule using HTML table elements.

## 📌 Project Overview

This project is designed to create a structured weekly timetable that displays activities, classes, and study sessions across weekdays.

It serves as a beginner-friendly project for learning HTML table structures and organizing information in a clear and readable format.

## 🚀 Features

* Displays a weekly schedule from Monday to Friday
* Organized using HTML table elements
* Includes time slots for different activities
* Uses semantic HTML tags for better structure
* Beginner-friendly and easy to customize
* No additional libraries or frameworks required

## 🛠️ Technologies Used

* HTML5
