Timetable Project (HTML Only)

This is a simple weekly timetable created using only HTML.

How to use:
1. Open the file timetable.html in your browser.
2. Edit the table in VS Code to add your own classes or activities.
3. Save the file and refresh your browser to see the changes.

Features:
- Days as columns and time slots as rows
- Uses basic HTML <table>
- Easy to customize
